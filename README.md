# 🕹️ Totally Rad 80's Trivia

A neon-soaked, single-file 80's trivia game you can play in any web browser. No server, no build step, no dependencies — just open `index.html`.

## Play

```bash
open index.html
```

Or play it live via GitHub Pages once enabled: **Settings → Pages → Deploy from branch → main / root**.

## Features

- 🎲 10 random questions per round from a 40-question bank
- 🗂️ Six categories: Movies, Music, TV, Games & Tech, Fads & Culture, History & World
- ⏱️ 15-second timer per question with a speed bonus (faster = more points)
- ✅ Instant feedback with the correct answer
- 🏆 End-of-round score and an 80's "rank" (from *Bogus run* to *80's Legend*)
- ⌨️ Keyboard support: `1-4` / `A-D` to answer, `Enter` / `Space` to advance

## Scoring

- 100 base points per correct answer
- Up to +75 speed bonus based on time remaining

Built with plain HTML, CSS, and vanilla JavaScript.
